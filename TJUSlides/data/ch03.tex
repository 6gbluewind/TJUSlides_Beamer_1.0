\subsection[SVM \& RF]{SVM \& RF}\label{subsec:3-1}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{frame}
  \frametitle{SVM}
  支持向量机(Support Vector Machines,SVM)是一种线性和非线性数据分析的分类方法，如线性SVM，非线性SVM，C-SVM(软间隔支持向量机)，$\nu$ -SVM，SVR。
  \begin{definition}
    考虑训练集$TR=\{(\mathbf{x}_{i},y_{i}),i=1,\ldots,l\}$，其中$\mathbf{x}_{i}\in\mathbb{R}^d$,$y_{i}\in\{+1,-1\}$。 如果存在$\mathbf{w}\in\mathbb{R}^d$和正数$\varepsilon$，使得对所有正类的下标$i$，有$\mathbf{w}\cdot x_{i} + b \geq\varepsilon$，而对所有负类下标$i$，有
    $\mathbf{w}\cdot x_{i} + b \leq -\varepsilon$，则称训练集$TR$ 线性可分。
  \end{definition}
\end{frame}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{frame}
  \frametitle{SVM}
  在近似线性可分的情形下，线性分类支持向量机的决策函数可描述为：
\begin{equation}\label{decisionfunction}
  f(\mathbf{x})=sgn(\mathbf{w}^{*}\cdot\mathbf{x}+b^{*})=sgn(\sum^{l}_{i=1}\alpha^{*}_{i}y_{i}\mathbf{x}_{i}\cdot\mathbf{x}+b^{*})
\end{equation}
\end{frame}

\subsection[Bioconductor]{Bioconductor}\label{subsec:3-2}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{frame}
  \frametitle{Bioconductor}
  \small
  Bioconductor用结构化的数据来表示生物概念。以慢性淋巴细胞白血病为例，使用聚类方法和主成分分析方法分类观察结果。根据由相关系数矩阵导出的距离矩阵进行聚类。稳定组和恶化分组没能很好分开，这是因为如果总体上两组数据是分开的，那么说明是我们关心的因素起主导作用；如果不是，则可能有其他的主导作用。
